# Website Templates

Professional website templates for local businesses. Built with pure HTML, CSS, and JavaScript. Free hosting on Cloudflare Pages.

## Templates

- **Classic Dark** (`alfredo-original/`) - Bold black and red, professional, no-nonsense
- **4-Style Sampler** (`alfredo-style-previews/`) - Bold Industrial, Clean Modern, Warm Approachable, High Energy
- **Scroll Animations** (`alfredo-animation/`) - CSS-powered scroll effects, particles, counters
- **GSAP Premium** (`alfredo-gsap/`) - Professional GSAP ScrollTrigger animations, horizontal scroll, parallax

## Features

- Mobile-first responsive design
- Click-to-call and directions buttons
- Google reviews integration
- SEO structured data (schema.org)
- FAQ section with FAQ schema markup
- Se habla espanol support
- Free hosting on Cloudflare Pages
- Google PageSpeed score 95-100

## Deployment

Push to GitHub, connect to Cloudflare Pages, deploy. Each folder becomes its own URL path.
